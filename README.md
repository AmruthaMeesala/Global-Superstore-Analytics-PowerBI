# Global Superstore Analytics Dashboard

## Project Overview

This project analyzes Global Superstore sales data using Power BI to identify trends in sales, profit, customer behavior, and regional performance.

The dashboard helps businesses monitor key performance indicators and make data-driven decisions.

## Tools Used

* Power BI
* DAX
* Excel

## Dataset

* Global Superstore Dataset
* 10,000+ Records
* Multiple Markets and Regions

## Key KPIs

* Total Sales
* Total Profit
* Profit Margin
* Total Orders
* Sales YTD
* Profit YTD

## Dashboard Features

### Sales Analysis

Tracks sales performance across categories, regions, and markets.

### Profit Analysis

Identifies profitable and loss-making products and regions.

### Customer Analysis

Highlights top customers and customer contribution.

### Geographic Analysis

Interactive map visualizations for regional performance.

### Product Analysis

Evaluates category and sub-category performance.

## Key Insights

* Technology category generated the highest sales.
* APAC market contributed significantly to revenue.
* Certain products generated high sales but low profit margins.
* Sales performance varied considerably across regions.
